# Chinese Chess Game

## Introduction
This program is a Chinese Chess game developed using object-oriented programming techniques and the Pygame library. It demonstrates basic concepts of object-oriented programming, from creating simple classes to their application.

## How to Run
1. Navigate to the directory containing the main.py file using the terminal or command prompt.

cd path/to/your/directory

2. Run the main.py file.

python main.py

## Game Rules
The game follows the rules of Chinese Chess. You can move pieces on the board and attempt to checkmate the opponent's king.
